App registration

Register your app to get an App ID and start using Deriv API for personal or business purposes.

Get started in 3 simple steps:

  1. Sign up for a Deriv account (demo or real).
  2. Create an admin-level access API token.
  3. Fill in the details below.

Your token (Admin level)

Your app name (required)

If you'd like to monetise your app, please complete the following sections.

Commission

For each trade performed on your app, you will receive a commission. Set a markup percentage below to determine the commission you will make. You can find more details on how the commission is calculated here.

Markup percentage (0.00 – 5.00%)

Authorisation

Give your clients authorised access to your app using one of the following:

  1. A static token

    Clients can generate API tokens by logging into their Deriv account and going to Manage account settings > API token.
    Recommended for personal trading. It is the easiest method of providing authorised access but less secure than Oauth.

  2. Oauth

    A more secure method of granting authorised access, but is also more complex to set up.
    Recommended for third party access (clients).

To use Oauth, please fill out the following fields. These details can be changed later using the app_update API call.

Website URL*

*Please note that this URL will be used as the Oauth redirect URL for the Oauth authorisation

Verification URL

Oauth authorisation levels

Please select the level of access you would like clients to give to your app.

Bear in mind that you generally need only 'Trade' and 'Trading information' access. ‘Admin’ access is usually not required.
